# 보상 페이지 - Ecostep

## 개요
보상 페이지는 포인트를 사용하여 물고기, 장식품을 구매하고 랭크 보상을 받는 화면입니다.

---

## 주요 기능

### 1. 랭크 시스템
**질문: 랭크는 어떤 종류가 있나요?**
- 브론즈 (Bronze): 0~2,099 포인트
- 실버 (Silver): 2,100~6,299 포인트
- 골드 (Gold): 6,300~12,599 포인트
- 플래티넘 (Platinum): 12,600 포인트 이상

**질문: 현재 랭크는 어디서 확인하나요?**
- 보상 페이지 상단에 큰 직사각형으로 표시됩니다
- 랭크 이름과 진행도가 표시됩니다

**질문: 랭크는 어떻게 올리나요?**
- 챌린지를 완료하여 포인트를 획득합니다
- 누적 포인트가 일정 수준에 도달하면 자동으로 랭크가 올라갑니다

**질문: 랭크 진행도는 무엇인가요?**
- 다음 랭크까지 남은 진행도를 퍼센트로 보여줍니다
- 예: "실버까지 45% 남음"
- 진행바로 시각적으로 확인할 수 있습니다
- 진행바는 현재 랭크 색상으로 표시됩니다
- 플래티넘 최고 등급에서는 "플래티넘 최고 등급 달성!" 메시지가 표시됩니다
- 플래티넘 등급의 진행률은 소수점 1자리까지 표시됩니다

**질문: 랭크 표시에서 별 효과는 무엇인가요?**
- 랭크 박스에는 반짝이는 별 애니메이션이 있습니다
- 애니메이션 별 5개와 정적 별 15개가 표시됩니다
- 별 효과로 랭크의 고급스러움을 표현합니다
- 페이지가 활성화되어 있을 때만 애니메이션이 작동합니다

**질문: 랭크 아이콘은 어떻게 표시되나요?**
- 각 랭크마다 고유한 아이콘이 있습니다:
  - 브론즈: 파란색 메달
  - 실버: 녹색/청록색 메달
  - 골드: 노란색 메달
  - 플래티넘: 보라색/분홍색 메달
- 하단에는 4개 랭크가 모두 표시됩니다
- 달성한 랭크는 컬러로, 미달성 랭크는 회색으로 표시됩니다
- 현재 랭크는 그라데이션 텍스트로 강조됩니다

### 2. 랭킹 보상 - 어항
**질문: 랭킹 보상은 무엇인가요?**
- 실버 어항: 실버 랭크 달성 시 수령 가능
- 골드 어항: 골드 랭크 달성 시 수령 가능
- 플래티넘 어항: 플래티넘 랭크 달성 시 수령 가능

**질문: 어항은 어떻게 받나요?**
- 해당 랭크에 도달하면 "수령 가능" 상태가 됩니다
- 어항 카드를 클릭하면 수령됩니다
- 수령한 어항은 자동으로 홈 화면에 적용됩니다

**질문: 이미 받은 어항은 어떻게 표시되나요?**
- "수령 완료"라고 표시됩니다
- 초록색 테두리가 생깁니다

**질문: 아직 받을 수 없는 어항은 어떻게 보이나요?**
- 잠금 아이콘이 표시됩니다
- 흐릿하게(블러) 처리됩니다
- 클릭하면 "○○ 랭크에서 잠금 해제" 메시지가 나타납니다

### 3. 물고기 구매
**질문: 물고기는 어떻게 구매하나요?**
- 보상 페이지에서 "물고기" 섹션을 찾습니다
- 원하는 물고기 카드를 클릭합니다
- 포인트가 충분하면 자동으로 구매됩니다

**질문: 물고기 가격은 얼마인가요?**
브론즈 랭크:
- 1번째 물고기: 200P
- 2번째 물고기: 300P
- 3번째 물고기: 400P

실버 랭크:
- 1번째 물고기: 500P
- 2번째 물고기: 600P
- 3번째 물고기: 700P

골드 랭크:
- 1번째 물고기: 1,000P
- 2번째 물고기: 1,100P
- 3번째 물고기: 1,200P

플래티넘 랭크:
- 1번째 물고기: 1,500P
- 2번째 물고기: 1,600P
- 3번째 물고기: 1,700P

**질문: 물고기를 구매하면 어디서 볼 수 있나요?**
- 홈 화면의 어항에 자동으로 추가됩니다
- 어항 설정에서 표시 여부를 선택할 수 있습니다

**질문: 포인트가 부족하면 어떻게 되나요?**
- "포인트 부족 (○○P 필요)" 메시지가 나타납니다
- 챌린지를 완료하여 포인트를 더 획득하세요

### 4. 장식품 구매
**질문: 장식품은 어떻게 구매하나요?**
- 보상 페이지에서 "장식품" 섹션을 찾습니다
- 원하는 장식품 카드를 클릭합니다
- 포인트가 충분하면 자동으로 구매됩니다

**질문: 장식품 가격은 어떻게 되나요?**
- 랭크별로 가격이 다릅니다
- 각 장식품 카드에 포인트가 표시됩니다

**질문: 장식품을 구매하면 어디서 사용하나요?**
- 홈 화면의 어항 설정에서 배치할 수 있습니다
- 더블클릭으로 위치, 크기, 회전을 조절할 수 있습니다

### 5. 랭크별 잠금 시스템
**질문: 잠금된 아이템은 무엇인가요?**
- 현재 랭크보다 높은 랭크의 아이템은 잠금 상태입니다
- 예: 브론즈 랭크에서는 실버, 골드, 플래티넘 아이템이 잠김

**질문: 잠금된 아이템은 어떻게 해제하나요?**
- 해당 랭크에 도달하면 자동으로 해제됩니다
- 예: 실버 물고기를 구매하려면 실버 랭크 이상이어야 합니다

**질문: 잠금된 아이템을 클릭하면 어떻게 되나요?**
- "○○ 랭크에서 잠금 해제" 메시지가 나타납니다
- 구매할 수 없습니다

## 자주 묻는 질문 (FAQ)

**Q: 포인트는 어떻게 얻나요?**
A: 챌린지를 완료하면 포인트를 얻을 수 있습니다. 매일 환경 팁 확인, 플라스틱 사용 줄이기 등의 챌린지가 있습니다.

**Q: 구매한 아이템을 취소할 수 있나요?**
A: 현재는 구매 취소 기능이 없습니다. 신중하게 선택해주세요.

**Q: 랭크가 내려갈 수 있나요?**
A: 아니요, 한 번 올라간 랭크는 내려가지 않습니다.

**Q: 모든 물고기와 장식품을 구매하려면 얼마나 필요한가요?**
A: 랭크별로 다르며, 모든 아이템을 구매하려면 상당한 포인트가 필요합니다. 꾸준히 챌린지를 완료하세요.

**Q: 플래티넘 랭크 이후에는 어떻게 되나요?**
A: 플래티넘은 최고 랭크입니다. 계속 포인트를 모아 더 많은 아이템을 구매할 수 있습니다.

**Q: 어항 보상을 받지 않으면 어떻게 되나요?**
A: 언제든지 받을 수 있습니다. 받지 않아도 랭크는 유지됩니다.

**Q: 구매 내역은 어디서 확인하나요?**
A: 구매한 아이템은 "구매완료" 표시가 되고, 홈 화면의 어항 설정에서도 확인할 수 있습니다.

## 팁

1. **우선순위 정하기**: 먼저 좋아하는 물고기나 장식품을 구매하세요
2. **랭크 올리기**: 높은 랭크에 도달하면 더 다양한 아이템을 구매할 수 있습니다
3. **포인트 절약**: 정말 원하는 아이템에만 포인트를 사용하세요
4. **어항 꾸미기**: 다양한 조합으로 나만의 어항을 만들어보세요
