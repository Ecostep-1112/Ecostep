# 기타 페이지 - Ecostep

## 개요
기타 페이지는 환경 팁, 주변 친환경 장소 검색, 고객 지원 등 추가 기능을 제공하는 화면입니다.

---

## 주요 기능

### 1. 오늘의 환경 상식
**질문: 오늘의 환경 상식은 무엇인가요?**
- 매일 새로운 환경 관련 팁과 정보를 제공합니다
- AI가 생성한 유용한 환경 보호 정보입니다
- 하루에 한 번씩 업데이트됩니다

**질문: 환경 상식 카테고리는 어떤 것들이 있나요?**
- **랜덤**: 모든 카테고리에서 무작위로 선택
- **재활용 팁**: 재활용 방법, 분리수거 관련 정보
- **생활 습관**: 일상에서 실천할 수 있는 환경 보호 습관
- **에너지 절약**: 전기, 물 등 에너지 절약 방법
- **제로웨이스트**: 쓰레기 줄이기, 플라스틱 대체 방법

**질문: 카테고리는 어떻게 변경하나요?**
1. 환경 상식 섹션의 카테고리 드롭다운 버튼을 클릭합니다
2. 드롭다운 메뉴에서 원하는 카테고리를 선택합니다:
   - 랜덤, 재활용 팁, 생활 습관, 에너지 절약, 제로웨이스트
3. 선택한 카테고리는 **내일부터** 적용됩니다
4. "내일부터 '○○' 팁이 표시됩니다" 성공 메시지가 표시됩니다
5. 오늘은 기존 카테고리의 팁이 계속 표시됩니다

**질문: 카테고리 변경이 바로 적용되지 않는 이유는?**
- 팁은 매일 자정에 생성됩니다
- 오늘 이미 생성된 팁을 보호하기 위함입니다
- 내일 자정이 되면 선택한 카테고리의 팁으로 변경됩니다
- 카테고리 변경은 다음 날부터 적용됩니다

**질문: 환경 팁을 확인하면 포인트를 받나요?**
- 네, 하루에 한 번 확인하면 **100 포인트**를 받습니다
- 팁 내용을 펼친 후 "확인(+100P)" 버튼을 클릭합니다
- 이미 확인한 경우 "확인 완료" 표시가 됩니다

**질문: 환경 팁을 공유할 수 있나요?**
- 네, "공유하기" 버튼을 클릭하면 됩니다
- 카카오톡, 다른 앱으로 공유하거나 클립보드에 복사할 수 있습니다

**질문: 환경 팁이 업데이트 안 돼요**
- 환경 팁은 매일 자정에 새로 업데이트됩니다
- 앱을 재시작하거나 페이지를 새로고침해보세요

### 2. 제로웨이스트 맵
**질문: 제로웨이스트 맵은 무엇인가요?**
- 주변의 친환경 장소를 찾아주는 기능입니다
- 위치 기반으로 3km 반경 내의 장소를 검색합니다
- 네이버 지도를 통해 장소를 찾을 수 있습니다

**질문: 어떤 장소를 검색할 수 있나요?**
- **제로웨이스트샵**: 포장 없이 제품을 판매하는 가게
- **리필스테이션**: 샴푸, 세제 등을 리필할 수 있는 곳
- **친환경매장**: 친환경 제품을 판매하는 매장
- **재활용센터**: 재활용품을 수거하는 센터

**질문: 위치 권한을 허용해야 하나요?**
- 네, 제로웨이스트 맵 기능을 사용하려면 위치 권한이 필요합니다
- 설정에서 "위치 공유" 옵션을 켜야 합니다
- 위치 권한이 없으면 "위치 설정을 켜면 주변 장소를 확인할 수 있습니다" 메시지가 표시됩니다

**질문: 장소 카테고리는 어떻게 변경하나요?**
1. 제로웨이스트 맵 섹션의 카테고리 드롭다운 버튼을 클릭합니다
2. 원하는 카테고리를 선택합니다:
   - 제로웨이스트샵, 리필스테이션, 친환경매장, 재활용센터
3. 즉시 해당 카테고리의 장소가 검색됩니다
4. 드롭다운 메뉴가 자동으로 닫힙니다

**질문: 장소 검색은 얼마나 자주 실행되나요?**
- 같은 카테고리와 위치에서는 10분간 이전 결과를 사용합니다
- 10분이 지나면 자동으로 새로 검색됩니다
- 카테고리를 변경하면 즉시 새로 검색됩니다
- 위치가 크게 변경되면 (약 100m 이상) 새로 검색됩니다

**질문: 장소 검색은 어떻게 이루어지나요?**
- 각 카테고리마다 여러 검색어로 검색합니다:
  - 제로웨이스트샵: "제로웨이스트샵", "제로웨이스트"
  - 리필스테이션: "리필스테이션", "리필샵"
  - 친환경매장: "친환경매장", "친환경제품"
  - 재활용센터: "재활용센터", "재활용"
- 모든 검색 결과를 합친 후 중복을 제거합니다
- 3km 반경 내 장소만 필터링합니다
- 거리 순으로 정렬됩니다
- 최대 20개 장소를 표시합니다

**질문: 검색된 장소 정보는 무엇인가요?**
- 장소 이름
- 주소
- 거리 (m 또는 km)

**질문: 장소로 이동하는 방법은?**
**모바일:**
1. 원하는 장소의 "이동" 버튼을 클릭합니다
2. 네이버 지도 앱으로 연결을 시도합니다
3. 앱이 설치되어 있으면 앱이 열립니다
4. 앱이 없으면 네이버 지도 웹사이트가 열립니다
5. 웹사이트에서 장소 검색 결과를 볼 수 있습니다

**PC:**
1. 원하는 장소의 "이동" 버튼을 클릭합니다
2. 네이버 지도 웹사이트가 새 탭에서 열립니다
3. 장소 검색 결과가 표시됩니다

**검색 방식:**
- 주소가 있으면 주소로 검색합니다
- 주소가 없으면 장소 이름으로 검색합니다

**질문: 장소가 검색되지 않아요**
- 3km 반경 내에 해당 장소가 없을 수 있습니다
- "3km 반경 내에 장소가 없습니다" 메시지가 표시됩니다
- 다른 카테고리를 선택해보세요

**질문: 검색 오류가 발생했어요**
- "장소를 불러오는데 실패했습니다" 메시지가 표시됩니다
- "다시 시도" 버튼을 클릭하여 재검색하세요
- 인터넷 연결을 확인하세요

### 3. 도움말
**질문: 도움말에는 무엇이 있나요?**
- **앱 사용 방법**: Coming Soon (준비 중)
- **고객 센터**: 챗봇을 통해 문의할 수 있습니다

**질문: 고객 센터는 어떻게 이용하나요?**
1. "고객 센터" 버튼을 클릭합니다
2. 챗봇 화면이 열립니다
3. 질문을 입력하면 AI 챗봇이 답변합니다

**질문: 챗봇에게 무엇을 물어볼 수 있나요?**
- 앱 사용 방법
- 기능 설명
- 문제 해결
- 환경 관련 정보

## 자주 묻는 질문 (FAQ)

**Q: 환경 팁 포인트를 못 받았어요**
A: 하루에 한 번만 포인트를 받을 수 있습니다. 이미 확인했다면 "확인 완료" 상태가 됩니다.

**Q: 카테고리를 변경했는데 팁이 안 바뀌어요**
A: 카테고리 변경은 내일부터 적용됩니다. 오늘은 기존 카테고리의 팁이 표시됩니다.

**Q: 위치 권한을 허용했는데도 장소가 안 나와요**
A: 설정에서 "위치 공유" 옵션이 켜져 있는지 확인하세요. 앱을 재시작해보세요.

**Q: 제로웨이스트샵이 3km 내에 없어요**
A: 다른 카테고리(리필스테이션, 친환경매장, 재활용센터)를 검색해보세요.

**Q: 네이버 지도가 실행되지 않아요**
A: 네이버 지도 앱이 설치되어 있지 않으면 웹 브라우저에서 열립니다.

**Q: 환경 팁을 다시 보고 싶어요**
A: 팁 제목을 클릭하면 내용을 다시 볼 수 있습니다.

**Q: 챗봇이 답변을 못 해요**
A: 질문을 다르게 표현해보거나, 더 구체적으로 질문해보세요.
