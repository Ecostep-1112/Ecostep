# 커뮤니티 페이지 - Ecostep

## 개요
커뮤니티 페이지는 친구들과 함께 환경 보호 활동을 비교하고 경쟁할 수 있는 화면입니다.

---

## 주요 기능

### 1. 친구 초대
**질문: 친구를 초대하려면 어떻게 하나요?**
커뮤니티 페이지 상단의 초대 버튼을 사용하세요:
- **카톡 버튼**: 카카오톡으로 초대 메시지 전송
- **링크 버튼**: 초대 링크를 클립보드에 복사
- **검색 버튼**: 아이디로 친구 검색

**질문: 카톡으로 초대하는 방법은?**
**모바일 (앱):**
1. "카톡" 버튼을 클릭합니다
2. 네이티브 공유 기능이 실행됩니다
3. 공유할 앱(카카오톡, 문자 등)을 선택합니다
4. 친구를 선택하여 초대 메시지를 전송합니다

**웹 (PC/모바일 브라우저):**
1. "카톡" 버튼을 클릭합니다
2. 카카오톡 공유 화면이 열립니다
3. 친구를 선택하여 초대 메시지를 전송합니다
4. 초대 메시지에는 앱 설명과 링크가 포함됩니다

**공통:**
- 친구가 링크를 클릭하여 가입하면 자동으로 친구가 됩니다
- 초대 코드는 내 아이디로 자동 생성됩니다

**질문: 링크로 초대하는 방법은?**
1. "링크" 버튼을 클릭합니다
2. 초대 링크가 자동으로 클립보드에 복사됩니다
3. "링크가 복사되었습니다" 메시지가 표시됩니다
4. 원하는 곳에 붙여넣기(Ctrl+V 또는 길게 터치)하여 공유합니다
5. 링크 형식: `https://ecostep.app/invite?code=[내아이디]`

**질문: 공유 기능이 작동하지 않으면?**
- 웹 환경에서는 Web Share API를 사용합니다
- 카카오톡 SDK가 초기화되지 않은 경우 클립보드 복사로 대체됩니다
- 클립보드 권한이 필요할 수 있습니다
- 모바일에서는 네이티브 공유 기능을 사용합니다

**질문: 초대 코드는 무엇인가요?**
- 초대 링크에 포함된 고유 코드입니다
- 내 아이디가 초대 코드로 사용됩니다
- 설정에서 아이디를 먼저 설정해야 합니다

### 2. 친구 검색
**질문: 친구를 검색하려면?**
1. "검색" 버튼을 클릭합니다
2. 검색 화면에서 친구의 아이디를 입력합니다
3. 검색 결과에서 친구를 찾아 친구 추가합니다

**질문: 아이디는 어디서 설정하나요?**
- 기타(More) 페이지의 설정에서 아이디를 설정할 수 있습니다
- 아이디를 설정해야 친구 초대와 검색 기능을 사용할 수 있습니다

### 3. 친구 랭킹
**질문: 친구 랭킹은 무엇인가요?**
- 나와 친구들의 플라스틱 절약량을 비교한 순위입니다
- 플라스틱 절약량이 많을수록 순위가 높습니다

**질문: 친구 랭킹은 어떻게 표시되나요?**
- 상위 3명: 메달 아이콘으로 표시
  - 1위: 플래티넘 메달 (28px, 가장 큼)
  - 2위: 골드 메달 (26px, 중간)
  - 3위: 실버 메달 (24px, 작음)
- 4위 이하: 동그란 테두리 안에 숫자로 순위 표시
- 내 순위는 테두리 색상이 다르게 강조됩니다
- 이름 아래에 아이디(@아이디)가 작게 표시됩니다
- 순위가 높을수록 글자 크기가 커집니다:
  - 1위: 14px
  - 2위: 13px
  - 3위: 12px
  - 4위 이하: 11px

**질문: 친구 랭킹에서 내가 4위 이하이면?**
- 상위 3명 아래에 구분선이 표시됩니다
- 구분선 아래에 내 순위가 별도로 표시됩니다
- 내 이름은 "나"로 표시됩니다
- 내 플라스틱 절약량과 순위가 함께 표시됩니다

**질문: 친구 랭킹에서 무엇을 볼 수 있나요?**
- 친구 이름
- 친구 아이디 (@아이디)
- 플라스틱 절약량 (g 또는 kg)
- 순위

**질문: 친구 랭킹 전체 목록은 어디서 보나요?**
- "더보기" 버튼을 클릭하면 전체 친구 목록을 볼 수 있습니다

### 4. 전체 랭킹
**질문: 전체 랭킹은 무엇인가요?**
- 모든 사용자의 플라스틱 절약량 순위입니다
- 전국 단위로 내 위치를 확인할 수 있습니다

**질문: 전체 랭킹 표시 방식은?**
- 상위 3명: 메달 아이콘으로 표시
  - 1위: 플래티넘 메달
  - 2위: 골드 메달
  - 3위: 실버 메달
- 내 순위가 4위 이하인 경우:
  - 상위 퍼센트로 표시 (예: "상위 15%")
  - 내 순위 번호 표시

**질문: 전체 랭킹에서 나는 어디 있나요?**
- 상위 3위 안에 들면: 메달과 함께 표시
- 4위 이하인 경우: 하단에 별도로 내 순위가 표시됩니다
- 순위가 100위 이상이면 "···"로 표시됩니다
- 상위 퍼센트가 함께 표시됩니다 (예: "상위 15%")
- 상위 퍼센트 계산: (내 순위 ÷ 전체 사용자 수) × 100

**질문: 상위 퍼센트는 어떻게 계산되나요?**
- 계산 공식: (순위 ÷ 전체 사용자 수) × 100
- 반올림하여 정수로 표시됩니다
- 예시:
  - 100명 중 10위 → 상위 10%
  - 500명 중 75위 → 상위 15%
  - 1000명 중 250위 → 상위 25%
- 1~3위는 메달로 표시되므로 퍼센트가 표시되지 않습니다

**질문: 전체 랭킹 전체 목록은 어디서 보나요?**
- "더보기" 버튼을 클릭하면 상위 50명의 전체 목록을 볼 수 있습니다

### 5. 플라스틱 절약량 표시
**질문: 플라스틱 절약량은 어떻게 표시되나요?**
- 1,000g 미만: g 단위 (예: 850g)
- 1,000g 이상: kg 단위 (예: 2.5kg)

**질문: 플라스틱 절약량은 어떻게 올리나요?**
- 챌린지를 완료하면 플라스틱 절약량이 증가합니다
- 꾸준히 챌린지를 완료할수록 순위가 올라갑니다

## 자주 묻는 질문 (FAQ)

**Q: 친구 초대를 했는데 친구 목록에 안 보여요**
A: 친구가 초대 링크를 통해 가입을 완료해야 친구 목록에 추가됩니다.

**Q: 아이디 설정을 안 하면 어떻게 되나요?**
A: "먼저 설정에서 아이디를 설정해주세요" 메시지가 나타나며 친구 초대 기능을 사용할 수 없습니다.

**Q: 내 순위가 표시되지 않아요**
A: 챌린지를 완료하여 플라스틱 절약량을 기록해야 랭킹에 표시됩니다.

**Q: 친구를 삭제할 수 있나요?**
A: 현재는 친구 삭제 기능이 제공되지 않습니다.

**Q: 랭킹은 실시간으로 업데이트되나요?**
A: 랭킹은 주기적으로 업데이트됩니다. 챌린지를 완료하면 반영됩니다.

**Q: 같은 플라스틱 절약량인 친구가 있으면 어떻게 되나요?**
A: 같은 절약량인 경우 먼저 달성한 사용자가 높은 순위를 받습니다.

**Q: 전체 랭킹 1위가 되면 특별한 혜택이 있나요?**
A: 현재는 별도 혜택이 없지만, 메달로 인정받을 수 있습니다.

## 팁

1. **친구와 경쟁**: 친구들과 플라스틱 절약량을 비교하며 동기부여를 받으세요
2. **꾸준한 챌린지**: 매일 챌린지를 완료하여 순위를 올리세요
3. **초대 보상**: 친구를 많이 초대하면 함께 환경을 보호할 수 있습니다
4. **목표 설정**: 전체 랭킹에서 상위 %를 목표로 설정해보세요
