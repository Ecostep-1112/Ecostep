# 챌린지 페이지 - Ecostep

## 개요
챌린지 페이지는 매일 환경 보호 활동을 실천하고 포인트를 획득하는 화면입니다.

---

## 주요 기능

### 1. 일일 챌린지
**질문: 일일 챌린지는 무엇인가요?**
- 매일 하나의 환경 보호 챌린지를 선택하여 실천합니다
- 챌린지를 완료하면 포인트를 획득합니다
- 플라스틱 절약량이 자동으로 기록됩니다

**질문: 챌린지는 어떻게 선택하나요?**
1. 챌린지 목록에서 원하는 챌린지를 선택합니다
2. "시작하기" 또는 "선택" 버튼을 클릭합니다
3. 선택한 챌린지가 하루 동안 유지됩니다

**질문: 챌린지 종류는 어떤 것들이 있나요?**
예시 챌린지:
- 일회용 컵 대신 텀블러 사용하기
- 비닐봉지 대신 에코백 사용하기
- 플라스틱 빨대 사용하지 않기
- 대중교통 이용하기
- 재활용 분리수거하기
- 종이 영수증 대신 전자 영수증 받기

**질문: 챌린지는 어떻게 완료하나요?**
1. 선택한 챌린지를 실천합니다
2. "완료" 버튼을 클릭합니다
3. 포인트와 플라스틱 절약량이 자동으로 추가됩니다

**질문: 챌린지 완료 시 포인트는 얼마나 받나요?**
- 일반 챌린지: 50~200 포인트
- 챌린지 종류와 난이도에 따라 포인트가 다릅니다

### 2. 주간 챌린지
**질문: 주간 챌린지는 무엇인가요?**
- 일주일(월~일) 동안 같은 챌린지를 반복합니다
- 매일 완료 여부가 표시됩니다
- 주간 진행률을 확인할 수 있습니다

**질문: 주간 챌린지 진행 상황은 어디서 보나요?**
- 챌린지 페이지 상단에 월~일 요일별 완료 현황이 표시됩니다
- 완료된 날: 녹색 체크 아이콘(✓)
- 미완료된 날: 빨간색 X 아이콘
- 아직 안 온 날: 회색 표시 또는 빈 칸
- 오늘: 테두리나 강조 표시로 구분됩니다

**질문: 챌린지를 완료하면 어떻게 되나요?**
- "완료" 버튼을 클릭합니다
- 100 포인트를 획득합니다
- 플라스틱 절약량이 자동으로 증가합니다 (챌린지마다 다름)
- 수질이 100%로 회복됩니다
- 오늘 날짜가 완료로 표시됩니다 (✓)
- 완료 후에는 "완료됨" 상태로 변경됩니다
- 챌린지 기록이 안전하게 저장됩니다

**질문: 챌린지를 선택하지 않고 완료할 수 있나요?**
- 아니요, 먼저 챌린지를 선택해야 합니다
- 선택하지 않고 완료 버튼을 누르면 "챌린지를 먼저 선택해주세요" 메시지가 표시됩니다
- 월요일에 챌린지를 선택하면 일주일 동안 유지됩니다

**질문: 주간 챌린지를 중간에 변경할 수 있나요?**
- 아니요, 한 번 챌린지를 시작하면 일주일 동안 변경할 수 없습니다
- 챌린지를 시작하면 (하루라도 완료하면) 챌린지 선택 화면이 비활성화됩니다
- 선택한 챌린지가 회색 배경에 가운데 정렬로 표시됩니다
- 월요일이 되면 새로운 챌린지를 선택할 수 있습니다

**질문: 월요일에는 어떻게 되나요?**
- 매주 월요일에 새로운 챌린지를 선택할 수 있습니다
- 이전 주의 챌린지는 히스토리에 저장됩니다
- 이전 주 달성률에 따라 보상을 받을 수 있습니다

**질문: 주간 챌린지 달성률은 어떻게 계산되나요?**
- 7일 중 완료한 날짜 수 ÷ 7 × 100
- 예: 5일 완료 시 약 71% 달성
- 70% 이상 달성 시 성공으로 표시됩니다

### 3. 플라스틱 사용 추적 (제로 챌린지)
**질문: 플라스틱 사용 추적은 무엇인가요?**
- 일주일 동안 사용한 플라스틱 양을 기록합니다
- 목표를 설정하고 달성 여부를 확인합니다
- 월요일마다 새로운 목표를 설정할 수 있습니다

**질문: 플라스틱 목표는 어떻게 설정하나요?**
1. "목표 설정" 버튼을 클릭합니다
2. 미리 정의된 목표 중 선택합니다:
   - 100g, 200g, 300g, 400g, 500g
   - 700g, 900g, 1100g, 1300g, 1500g
3. 직접 입력하여 커스텀 목표를 설정합니다:
   - 원하는 숫자를 입력합니다 (예: 250g)
   - "g" 또는 "kg" 단위를 사용할 수 있습니다 (예: 1kg = 1000g)
   - 사용자 정의 목표는 목록에 자동으로 추가됩니다
4. 목표는 일주일 동안 고정됩니다
5. 설정한 목표는 자동으로 저장됩니다

**질문: 목표를 여러 개 저장할 수 있나요?**
- 네, 커스텀 목표를 여러 개 저장할 수 있습니다
- 저장된 커스텀 목표는 목록에서 선택할 수 있습니다
- 저장된 목표를 삭제할 수도 있습니다
- 미리 정의된 목표와 커스텀 목표가 함께 표시됩니다

**질문: 플라스틱 사용량은 어떻게 기록하나요?**
1. "플라스틱 추가" 버튼을 클릭합니다
2. 사용한 플라스틱 품목을 선택합니다 (컵, 병, 봉지 등)
3. 개수를 입력합니다
4. 자동으로 무게가 계산되어 기록됩니다

**질문: 플라스틱 목표를 달성하면 어떻게 되나요?**
- 월요일에 지난 주 달성률을 확인합니다
- 달성률 계산: 100 - (사용량/목표 × 100)
- 달성률 1% 이상 시 알림이 발송됩니다
- 알림에서 보상 포인트(700P)를 수령할 수 있습니다
- 예시:
  - 목표 500g, 사용량 400g → 달성률 20% → 알림 발송
  - 목표 300g, 사용량 100g → 달성률 67% → 알림 발송
  - 목표 200g, 사용량 200g → 달성률 0% → 알림 없음

**질문: 플라스틱 사용량 통계는 어떻게 보나요?**
- 사용량 기간 드롭다운을 클릭하여 기간을 선택합니다:
  - **일주일**: 최근 7일간 일별 사용량 그래프
  - **한 달**: 최근 4주간 주별 사용량 그래프
  - **6개월**: 최근 6개월간 월별 사용량 그래프
  - **1년**: 최근 12개월간 월별 사용량 그래프
  - **전체**: 전체 기간 연도별 사용량 그래프 (최대 6년)
- 각 기간별로 그래프가 표시됩니다
- 날짜/주/월별 라벨이 표시됩니다
- 사용량은 g 또는 kg 단위로 표시됩니다

**질문: 그래프는 어떻게 읽나요?**
- 막대 그래프 형태로 표시됩니다
- 높이가 높을수록 사용량이 많습니다
- 최대값을 기준으로 상대적 높이가 계산됩니다
- 각 막대에 마우스를 올리면 정확한 값을 볼 수 있습니다

**질문: 플라스틱 목표는 언제 변경할 수 있나요?**
- 매주 월요일에만 목표를 변경할 수 있습니다
- 일주일 동안은 목표가 고정됩니다
- 변경 시도 시 "목표 변경은 월요일에 가능합니다" 메시지가 표시됩니다

### 4. 커스텀 챌린지
**질문: 커스텀 챌린지는 무엇인가요?**
- 사용자가 직접 만드는 챌린지입니다
- 자신만의 환경 보호 활동을 설정할 수 있습니다
- AI가 챌린지의 플라스틱 절약량을 자동으로 추정합니다

**질문: 커스텀 챌린지는 어떻게 만드나요?**
1. 챌린지 선택 화면에서 "기타 (직접 입력)" 선택
2. 챌린지 이름을 입력합니다:
   - 예: "대나무 칫솔 사용하기"
   - 예: "커피 찌꺼기 재활용하기"
   - 예: "친환경 세제 사용하기"
3. Enter 키를 누르거나 확인 버튼을 누릅니다
4. 자동으로 검증되어 챌린지 목록에 추가됩니다
5. 일반 챌린지처럼 선택하고 완료할 수 있습니다

**질문: 커스텀 챌린지의 절약량은 어떻게 계산되나요?**
- AI가 챌린지 내용을 분석하여 자동으로 추정합니다
- 플라스틱과 관련된 챌린지는 g 단위로 절약량이 표시됩니다
- 플라스틱과 무관한 챌린지는 "플라스틱과 무관한 챌린지" 경고가 표시됩니다
- 너무 모호한 챌린지는 "구체적인 방법을 명시해주세요" 메시지가 표시됩니다

**질문: 커스텀 챌린지 포인트는 얼마인가요?**
- 기본적으로 100 포인트가 부여됩니다
- 플라스틱 절약량은 챌린지 내용에 따라 다릅니다
- 저장된 절약량은 자동으로 기록됩니다

**질문: 커스텀 챌린지가 중복되면 어떻게 되나요?**
- "이미 존재하는 챌린지입니다" 메시지가 표시됩니다
- 같은 이름의 챌린지는 추가할 수 없습니다
- 다른 이름으로 변경하여 다시 시도하세요

### 5. 챌린지 히스토리
**질문: 챌린지 히스토리는 무엇인가요?**
- 과거에 완료한 챌린지 기록을 확인할 수 있습니다
- 주간 단위로 챌린지 진행 상황이 표시됩니다

**질문: 챌린지 히스토리에서 무엇을 볼 수 있나요?**
- 챌린지 이름
- 시작일과 종료일
- 완료한 날짜 수
- 달성률 (%)
- 성공 여부

**질문: 히스토리 기간은 어떻게 설정하나요?**
- 기본적으로 최근 7일이 표시됩니다
- 드롭다운에서 기간을 선택할 수 있습니다:
  - 일주일
  - 4주
  - 16주
  - 32주

### 6. 플라스틱 품목 관리
**질문: 플라스틱 품목은 어떤 것들이 있나요?**
미리 정의된 품목 (카테고리별):

**음료 관련:**
- 플라스틱병 (500ml): 25g
- 일회용 컵 (카페): 10g
- 페트병(대) (1.5L): 45g
- 빨대: 1g

**봉투류:**
- 비닐봉지(소) (편의점): 3g
- 비닐봉지(대) (마트): 7g

**배달/음식 관련:**
- 음식용기 (배달용기): 35g
- 일회용 수저/포크 (세트): 3g
- 일회용 접시: 8g

**기타 생활용품:**
- 화장품 용기 (소형): 15g

**질문: 커스텀 플라스틱 품목을 추가할 수 있나요?**
1. "커스텀 품목 추가" 버튼을 클릭합니다
2. 품목 이름을 입력합니다
3. 무게를 입력합니다 (g 단위)
4. 저장하면 품목 목록에 추가됩니다

**질문: 플라스틱 무게는 어떻게 계산되나요?**
- 각 품목마다 평균 무게가 설정되어 있습니다
- 개수를 입력하면 자동으로 총 무게가 계산됩니다
- 예: 플라스틱 컵 1개 = 약 10g

## 자주 묻는 질문 (FAQ)

**Q: 챌린지를 완료하는 것을 잊었어요**
A: 자정이 지나면 해당 날짜는 미완료로 표시됩니다. 다음 날부터 다시 완료할 수 있습니다.

**Q: 챌린지를 중간에 변경할 수 있나요?**
A: 주간 챌린지는 월요일에만 변경할 수 있습니다. 일주일 동안 같은 챌린지를 유지하세요.

**Q: 플라스틱 목표를 달성하지 못하면 어떻게 되나요?**
A: 달성하지 못해도 페널티는 없습니다. 다음 주에 더 낮은 목표를 설정해보세요.

**Q: 챌린지 완료 포인트가 지급되지 않았어요**
A: 챌린지를 완료하면 자동으로 포인트가 지급됩니다. 포인트 내역을 확인해보세요.

**Q: 주간 챌린지를 70% 달성하면 무엇을 받나요?**
A: 70% 이상 달성 시 히스토리에 "성공"으로 표시되며, 포인트를 획득합니다.

**Q: 커스텀 챌린지를 삭제할 수 있나요?**
A: 현재는 삭제 기능이 제공되지 않습니다.

**Q: 플라스틱 기록을 수정할 수 있나요?**
A: 현재는 수정 기능이 제공되지 않습니다. 신중하게 입력하세요.

**Q: 일요일에 챌린지를 완료하지 못하면?**
A: 월요일에 새로운 주가 시작되며, 이전 주 챌린지는 히스토리로 이동합니다.

## 팁

1. **일관성 유지**: 매일 챌린지를 완료하여 연속 기록을 유지하세요
2. **작은 목표부터**: 처음에는 쉬운 챌린지나 낮은 플라스틱 목표로 시작하세요
3. **커스텀 활용**: 자신만의 환경 보호 활동을 커스텀 챌린지로 만들어보세요
4. **히스토리 확인**: 과거 기록을 보며 진행 상황을 확인하세요
5. **월요일 알림**: 매주 월요일에 새로운 챌린지를 설정하는 것을 잊지 마세요
