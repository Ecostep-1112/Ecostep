# 홈 페이지 - Ecostep

## 개요
홈 페이지는 사용자의 메인 어항을 보여주는 화면입니다. 물고기, 장식품, 수질 상태를 한눈에 확인할 수 있습니다.

---

## 주요 기능

### 1. 어항 보기
**질문: 어항에서 무엇을 볼 수 있나요?**
- 구매한 물고기들이 어항 안에서 헤엄칩니다
- 배치한 장식품들이 표시됩니다
- 현재 수질 상태가 시각적으로 보입니다
- 물 표면 효과와 기포 애니메이션이 나타납니다

**질문: 어항 종류는 어떤 것들이 있나요?**
- 기본 어항 (브론즈 랭크)
- 실버 어항 (실버 랭크 보상)
- 골드 어항 (골드 랭크 보상)
- 플래티넘 어항 (플래티넘 랭크 보상)

### 2. 물고기 관리
**질문: 물고기는 어떻게 움직이나요?**
- 일부 물고기는 자동으로 좌우로 헤엄칩니다
- 움직이는 물고기 목록:
  - 코리도라스: 하층 선호, 느린 속도
  - 체리바브: 중층, 보통 속도
  - 네온테트라: 중층, 빠른 속도
  - 아피스토그라마: 중층, 매우 빠른 속도
  - 람시클리드: 중층, 보통 속도
  - 구피: 중층, 느린 속도
  - 엔젤피쉬: 중층, 느린 속도
  - 킬리피쉬: 중층, 빠른 속도
  - 베타: 중층, 느린 속도
  - 디스커스: 중층, 매우 느린 속도
  - 만다린피쉬: 중층, 보통 속도
  - 아로와나: 중층, 느린 속도 (더 넓은 공간 필요)
- 각 물고기는 좌우로 헤엄치며 벽에 닿으면 방향을 바꿉니다

**질문: 물고기의 위치는 어떻게 결정되나요?**
- 물고기는 어항 안에서 자연스럽게 배치됩니다
- 물고기 종류에 따라 선호하는 수심(상층/중층/하층)이 다릅니다
- 자동으로 고르게 분산되어 배치됩니다
- 물고기끼리 겹치지 않도록 자동 조정됩니다

**질문: 어항에 물고기를 추가하려면?**
- 어항 설정 버튼(톱니바퀴 아이콘)을 누릅니다
- "물고기 선택" 탭에서 원하는 물고기를 선택합니다
- "랜덤 표시" 옵션을 켜면 자동으로 물고기가 배치됩니다
- 물고기 수는 "물고기 개수" 슬라이더로 조절할 수 있습니다

**질문: 물고기가 너무 많으면 어떻게 하나요?**
- 어항 설정에서 물고기 개수를 줄일 수 있습니다
- 특정 물고기의 선택을 해제할 수 있습니다
- 랜덤 표시를 사용하면 자동으로 적절한 수가 표시됩니다

### 3. 장식품 배치 및 조정
**질문: 장식품은 어떻게 움직이나요?**
- 장식품을 더블클릭(PC)하거나 더블탭(모바일)하면 설정 패널이 나타납니다
- 설정 패널이 열린 상태에서 장식품을 터치하여 드래그하면 위치를 변경할 수 있습니다
- 어항 내부 어디든 장식품을 배치할 수 있습니다
- 드래그한 위치는 자동으로 저장됩니다

**질문: 장식품 설정 패널은 어떻게 열고 닫나요?**
- 더블클릭(PC) 또는 더블탭(모바일)으로 패널을 엽니다
- 패널이 열린 상태에서 같은 장식품을 다시 더블클릭하면 패널이 닫힙니다
- 패널의 "닫기" 버튼을 클릭해도 닫을 수 있습니다
- 어항의 빈 공간을 클릭하면 패널이 자동으로 닫힙니다
- 패널은 화면 하단에서 부드럽게 나타납니다

**질문: iOS에서 장식품 터치가 잘 안되는데요?**
- 더블탭 대신 두 번 빠르게 탭하세요
- 장식품을 정확히 터치해야 합니다
- 설정 패널이 열린 후에는 드래그가 가능합니다
- 조금만 이동하면 드래그로 인식됩니다

**질문: 장식품 크기와 회전은 어떻게 조절하나요?**
- 장식품을 더블클릭하여 설정 패널을 엽니다
- **크기 조절**: 슬라이더를 움직여 50%~150% 범위로 조절 가능
  - 현재 크기가 %로 표시됩니다
  - 실시간으로 크기가 변경됩니다
- **회전 조절**: 슬라이더를 움직여 0°~360° 범위로 회전 가능
  - 현재 각도가 °로 표시됩니다
  - 실시간으로 회전이 적용됩니다
- 조절한 값은 자동으로 저장됩니다

**질문: 장식품 설정을 초기화하려면?**
- 설정 패널의 "기본값으로 리셋" 버튼을 클릭합니다
- 크기는 100%, 회전은 0°로 초기화됩니다
- 위치는 초기화되지 않고 현재 위치를 유지합니다

**질문: 장식품 위치를 저장하려면?**
- 설정 패널에서 "장식품 위치 저장" 버튼을 누릅니다
- 드래그로 이동한 위치는 이미 자동으로 저장됩니다
- 위치 저장 버튼은 현재 설정을 확정할 때 사용합니다
- 저장 완료 시 "장식품 설정이 저장되었습니다" 메시지가 나타납니다

**질문: 장식품이 겹쳐 있으면 어떻게 하나요?**
- 장식품을 더블클릭하여 설정 패널을 엽니다
- 드래그하여 원하는 위치로 이동시킵니다
- 다른 장식품과 겹치지 않도록 조정할 수 있습니다
- 크기를 줄이면 더 많은 장식품을 배치할 수 있습니다

### 4. 수질 관리
**질문: 수질은 무엇인가요?**
- 수질은 어항의 깨끗한 정도를 나타냅니다
- 0%~100% 범위로 표시됩니다
- 수질이 낮으면 어항이 흐릿하게 보입니다

**질문: 수질은 어떻게 관리하나요?**
- 매일 챌린지를 완료하면 수질이 유지됩니다
- 챌린지를 하지 않으면 하루에 5%씩 감소합니다
- 수질 80% 이상: 맑음 (파란색)
- 수질 50~79%: 보통 (노란색)
- 수질 50% 미만: 탁함 (빨간색)

### 5. 환경 통계
**질문: 플라스틱 절약량은 어떻게 계산되나요?**
- 챌린지를 완료하면 플라스틱 절약량이 증가합니다
- 1,000g 미만: g 단위로 표시
- 1,000g 이상: kg 단위로 표시

**질문: 나무 효과는 무엇인가요?**
- 플라스틱 절약으로 인한 CO₂ 감소량을 나무로 환산한 것입니다
- 계산 공식:
  - 플라스틱 1kg 절약 = 약 3kg CO₂ 감소
  - 나무 1그루 = 연간 약 12kg CO₂ 흡수
  - 플라스틱 4kg 절약 = 나무 1그루의 연간 효과
- 예시:
  - 12kg 플라스틱 절약 → 36kg CO₂ 감소 → 나무 3그루 효과
- 나무 그림은 최대 7개씩 줄로 표시됩니다
- 나무 그루 수가 증가하면 여러 줄로 표시됩니다

**질문: 나무 아이콘이 어떻게 표시되나요?**
- 나무 아이콘은 플라스틱 절약량에 따라 자동으로 생성됩니다
- 각 나무는 애니메이션 효과가 있습니다
- 나무가 0그루이면 "챌린지를 완료하면 나무가 자랍니다" 메시지가 표시됩니다
- 나무 수가 많을수록 여러 줄로 표시됩니다

**질문: 연속 달성은 무엇인가요?**
- 챌린지를 연속으로 완료한 일수입니다
- 홈 화면 상단에 불꽃 아이콘과 함께 표시됩니다

### 6. 어항 설정
**질문: 어항 설정은 어디서 하나요?**
- 어항 하단의 톱니바퀴 버튼을 누릅니다
- 설정 패널이 화면 하단에서 나타납니다
- 설정 패널은 탭 형식으로 구성되어 있습니다

**질문: 어항 설정에서 무엇을 할 수 있나요?**
- **어항 테마 변경**: 랭크별 어항 선택
- **물고기 선택**: 구매한 물고기 중 표시할 물고기 선택
- **장식품 선택**: 구매한 장식품 중 표시할 장식품 선택
- **랜덤 표시 옵션**: 물고기 자동 배치 설정
- **물고기 개수 조절**: 슬라이더로 표시할 물고기 수 조절

**질문: 어항 테마는 어떻게 변경하나요?**
- 어항 설정 패널의 "어항" 탭을 선택합니다
- 보상 페이지에서 수령한 어항 중 선택할 수 있습니다
- 브론즈, 실버, 골드, 플래티넘 어항이 있습니다
- 아직 수령하지 않은 어항은 잠금 상태로 표시됩니다
- 선택하면 즉시 어항 디자인이 변경됩니다

**질문: 물고기 표시/숨기기는 어떻게 하나요?**
- 어항 설정 패널의 "물고기" 탭을 선택합니다
- 구매한 물고기 목록이 표시됩니다
- 각 물고기의 체크박스를 클릭하여 표시/숨기기를 설정합니다
- 체크된 물고기만 어항에 표시됩니다
- 변경사항은 자동으로 저장됩니다

**질문: 장식품 표시/숨기기는 어떻게 하나요?**
- 어항 설정 패널의 "장식품" 탭을 선택합니다
- 구매한 장식품 목록이 표시됩니다
- 각 장식품의 체크박스를 클릭하여 표시/숨기기를 설정합니다
- 체크된 장식품만 어항에 표시됩니다
- 장식품 위치는 이전에 설정한 위치로 표시됩니다

**질문: 랜덤 표시 옵션은 무엇인가요?**
- 물고기를 자동으로 배치하는 기능입니다
- 켜면: 선택된 물고기들이 무작위로 배치됩니다
- 끄면: 수동으로 물고기 위치를 조절할 수 있습니다 (구현 예정)
- 랜덤 표시 시 물고기끼리 겹치지 않도록 자동 조정됩니다

## 자주 묻는 질문 (FAQ)

**Q: 물고기가 움직이지 않아요**
A: 일부 물고기는 정지 상태입니다. 코리도라스, 네온테트라, 구피, 베타 등은 자동으로 움직입니다.

**Q: 장식품이 겹쳐 있어요**
A: 장식품을 더블클릭하여 드래그로 위치를 조정할 수 있습니다.

**Q: 어항이 흐릿해요**
A: 수질이 낮아졌습니다. 매일 챌린지를 완료하여 수질을 유지하세요.

**Q: 통계가 업데이트되지 않아요**
A: 챌린지를 완료하면 자동으로 업데이트됩니다.

**Q: 어항을 변경하고 싶어요**
A: 보상 페이지에서 랭크를 올려 새로운 어항을 받을 수 있습니다.
