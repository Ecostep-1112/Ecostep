## 1. 설정 접근

### 1. 설정 메뉴 접근
**질문: 설정은 어디서 열 수 있나요?**
- More(더보기) 페이지에서 접근할 수 있습니다
- 설정 아이콘 또는 설정 버튼을 클릭합니다

## 2. 프로필 설정

### 1. 내 정보 확인
**질문: 내 정보는 어디서 볼 수 있나요?**
- 설정 메뉴 상단의 프로필 섹션을 클릭합니다
- 프로필에는 다음 정보가 표시됩니다:
  - 프로필 사진 또는 기본 아바타
  - 이름
  - 아이디 (@아이디)

**질문: 프로필 사진은 어떻게 표시되나요?**
- 설정 메뉴: 10x10 크기의 원형 아바타
- 프로필 화면: 20x20 크기의 원형 아바타
- 이미지가 설정되어 있으면 이미지가 표시됩니다
- 이미지가 없으면 카메라 아이콘이 표시됩니다
- 설정 메뉴에서는 이름 또는 아이디의 첫 글자가 표시됩니다

**질문: 프로필을 수정하려면?**
- 프로필 섹션을 클릭하여 "내 정보" 화면으로 이동합니다
- 다음 정보를 수정할 수 있습니다:
  - 프로필 사진
  - 이름
  - 아이디
  - 생년월일
  - 휴대폰 번호
  - 이메일 주소

### 2. 프로필 사진 업로드
**질문: 프로필 사진은 어떻게 변경하나요?**
1. 프로필 화면 상단의 원형 프로필 사진을 클릭합니다
2. 오른쪽 하단에 작은 플러스(+) 아이콘이 표시됩니다
3. 플러스 아이콘을 클릭하면 파일 선택 창이 열립니다
4. 원하는 이미지 파일을 선택합니다 (JPG, PNG 등)
5. 선택한 이미지가 자동으로 업로드되고 표시됩니다

**질문: 프로필 사진은 어디에 저장되나요?**
- 안전하게 저장되며 다른 기기에서도 동기화됩니다
- 즉시 반영됩니다

**질문: 프로필 사진 크기 제한이 있나요?**
- 파일 크기 제한은 브라우저의 기본 제한을 따릅니다
- 이미지는 원형으로 잘려서 표시됩니다
- 정사각형 이미지를 사용하는 것이 좋습니다

### 3. 이름 수정
**질문: 이름은 어떻게 변경하나요?**
1. 프로필 화면에서 "이름" 항목을 클릭합니다
2. 이름 입력 화면이 열립니다
3. 새로운 이름을 입력합니다 (1~10자)
4. "저장" 버튼을 클릭합니다

**질문: 이름에 제한이 있나요?**
- 최소 1자, 최대 10자까지 입력 가능
- 한글, 영문, 숫자 모두 사용 가능
- 공백도 허용됩니다
- 빈 이름은 저장할 수 없습니다

**질문: 이름이 저장되지 않으면?**
- 1~10자 범위를 벗어났는지 확인하세요
- 빈 값은 저장할 수 없습니다
- 네트워크 연결을 확인하세요

### 4. 아이디 수정
**질문: 아이디는 어떻게 변경하나요?**
1. 프로필 화면에서 "아이디" 항목을 클릭합니다
2. 아이디 입력 화면이 열립니다
3. 새로운 아이디를 입력합니다 (1~15자)
4. "저장" 버튼을 클릭합니다

**질문: 아이디 규칙은 무엇인가요?**
- **사용 가능한 문자**: 영문 (a-z, A-Z), 숫자 (0-9), 언더스코어 (_)
- **길이**: 1~15자
- **중복 확인**: 다른 사용자가 사용 중인 아이디는 사용 불가
- **실시간 필터링**: 허용되지 않는 문자는 자동으로 제거됩니다

**질문: 아이디 중복 확인은 어떻게 되나요?**
- "저장" 버튼을 클릭하면 자동으로 중복 확인이 진행됩니다
- 이미 사용 중인 아이디면 "이미 사용 중인 아이디입니다" 메시지가 표시됩니다
- 사용 가능한 아이디면 즉시 저장됩니다

**질문: 아이디는 어디에 사용되나요?**
- 친구 초대 링크에 포함됩니다
- 커뮤니티 랭킹에 표시됩니다
- 다른 사용자가 나를 검색할 때 사용됩니다

**질문: 아이디 변경 중 오류가 발생하면?**
- "저장 중..." 메시지가 표시되는 동안 기다립니다
- 중복 클릭을 방지하기 위해 버튼이 비활성화됩니다
- 오류 발생 시 "아이디 저장에 실패했습니다" 메시지가 표시됩니다

### 5. 생년월일 수정
**질문: 생년월일은 어떻게 변경하나요?**
1. 프로필 화면에서 "생년월일" 항목을 클릭합니다
2. 생년월일 입력 화면이 열립니다
3. 년도, 월, 일을 각각 입력합니다
4. "저장" 버튼을 클릭합니다

**질문: 생년월일 입력 형식은?**
- **년도**: 4자리 또는 2자리
  - 2자리 입력 시 자동 변환:
    - 00~29 → 2000~2029년
    - 30~99 → 1930~1999년
  - 예: 95 → 1995, 05 → 2005
- **월**: 1~12
- **일**: 1~31 (월에 따라 자동 검증)
- **저장 형식**: YYYY.MM.DD (예: 1995.03.15)

**질문: 생년월일 유효성 검사는?**
- 년도: 1900년 ~ 올해
- 월: 1~12 (현재 년도인 경우 현재 월까지)
- 일: 각 월의 일수에 맞게 (윤년 자동 계산)
- 미래 날짜는 입력 불가
- 잘못된 입력 시 해당 필드가 빨간색 테두리로 표시됩니다

**질문: 생년월일을 지우려면?**
- 모든 필드를 비워두고 "저장"을 클릭합니다
- 빈 값으로 저장됩니다

**질문: 생년월일 입력 오류는?**
- 빨간색 테두리가 표시된 필드를 수정하세요
- 예시:
  - 월에 13 입력 → 빨간색 (1~12만 가능)
  - 2월에 30일 입력 → 빨간색 (2월은 최대 28/29일)
  - 미래 날짜 입력 → 빨간색

### 6. 휴대폰 번호 수정
**질문: 휴대폰 번호는 어떻게 변경하나요?**
1. 프로필 화면에서 "휴대폰 번호" 항목을 클릭합니다
2. 휴대폰 번호 입력 화면이 열립니다
3. 숫자만 입력합니다 (하이픈 자동 입력)
4. "저장" 버튼을 클릭합니다

**질문: 휴대폰 번호 형식은?**
- **입력 형식**: 숫자만 입력 (하이픈 불필요)
- **자동 포맷팅**: XXX-XXXX-XXXX 형식으로 자동 변환
- **예시**: 01012345678 입력 → 010-1234-5678로 표시
- **길이**: 10~11자리 (하이픈 제외)

**질문: 휴대폰 번호 유효성 검사는?**
- 10자리 (지역번호 포함) 또는 11자리 (휴대폰)
- 숫자만 입력 가능
- 빈 값도 허용됩니다
- 10~11자리가 아니면 "10~11자리 숫자를 입력해주세요" 에러 메시지

**질문: 휴대폰 번호를 지우려면?**
- 모든 숫자를 지우고 "저장"을 클릭합니다
- 빈 값으로 저장됩니다

### 7. 이메일 수정
**질문: 이메일은 어떻게 변경하나요?**
1. 프로필 화면에서 "이메일 주소" 항목을 클릭합니다
2. 이메일 입력 화면이 열립니다
3. 이메일 주소를 입력합니다
4. "저장" 버튼을 클릭합니다

**질문: 이메일 형식 검사는?**
- 기본 이메일 형식 검증: `example@domain.com`
- 이메일 형식이 올바른지 자동으로 검증됩니다
- 빈 값도 허용됩니다
- 잘못된 형식이면 저장되지 않습니다

**질문: 이메일을 지우려면?**
- 모든 텍스트를 지우고 "저장"을 클릭합니다
- 빈 값으로 저장됩니다

### 8. 계정 탈퇴
**질문: 계정을 탈퇴하려면?**
1. 프로필 화면 하단의 "계정 탈퇴" 버튼을 클릭합니다
2. 확인 다이얼로그가 나타납니다:
   - "정말 탈퇴하시겠습니까?"
   - "계정을 탈퇴하시면 모든 데이터가 영구적으로 삭제되며, 복구할 수 없습니다."
3. "네, 탈퇴합니다" 버튼을 클릭합니다
4. 계정이 삭제되고 로그인 화면으로 이동합니다

**질문: 계정 탈퇴 버튼은 어디 있나요?**
- 프로필 화면 하단에 고정되어 있습니다
- 내비게이션 바 바로 위 (64px 위)
- 빨간색 텍스트와 빨간색 테두리
- 편집 화면에서는 표시되지 않습니다

**질문: 탈퇴하면 데이터는 어떻게 되나요?**
- **모든 데이터가 영구 삭제됩니다**:
  - 프로필 정보 (이름, 아이디, 생년월일 등)
  - 챌린지 기록
  - 플라스틱 사용 기록
  - 친구 목록
  - 포인트 및 랭킹
  - 구매한 물고기/장식품
- **복구 불가능**: 삭제 후 복구할 수 없습니다

**질문: 탈퇴를 취소하려면?**
- 확인 다이얼로그에서 "아니요" 버튼을 클릭합니다
- 다이얼로그가 닫히고 탈퇴가 취소됩니다

### 9. 데이터 저장 및 동기화
**질문: 프로필 변경사항은 언제 저장되나요?**
- "저장" 버튼을 클릭하는 즉시 저장됩니다
- 안전하게 저장되며 다른 기기에서도 동기화됩니다
- 저장 완료 시 자동으로 편집 화면이 닫힙니다

**질문: 다른 기기에서도 동기화되나요?**
- 네, 안전하게 저장됩니다
- 다른 기기에서 로그인하면 모든 정보가 동기화됩니다
- 실시간 동기화는 아니며, 앱 재시작 시 반영됩니다

**질문: 저장 실패 시 어떻게 되나요?**
- 오류 메시지가 토스트로 표시됩니다
- "저장에 실패했습니다" 메시지가 표시됩니다
- 네트워크 연결을 확인하고 다시 시도하세요

## 3. 언어 설정

### 1. 언어 변경
**질문: 언어는 어떻게 변경하나요?**
1. 설정 메뉴에서 "언어" 항목을 클릭합니다
2. 원하는 언어를 선택합니다:
   - **한국어**: 사용 가능
   - **English**: Coming Soon (준비 중)

**질문: 영어는 왜 사용할 수 없나요?**
- 영어 버전은 현재 개발 중입니다
- "Coming Soon" 메시지가 표시됩니다
- 버튼이 비활성화되어 있으며 블러 효과가 적용됩니다

**질문: 언어 설정은 어떻게 표시되나요?**
- 선택된 언어는 청록색 테두리와 그라데이션 체크 아이콘으로 표시됩니다
- 설정 메뉴에는 현재 언어가 "한국어" 또는 "English"로 표시됩니다

## 4. 알림 설정

### 1. 알림 ON/OFF
**질문: 알림은 어떻게 설정하나요?**
1. 설정 메뉴에서 "알림" 항목을 클릭합니다
2. "알림 받기" 토글 스위치를 클릭하여 켜거나 끕니다

**질문: 알림 토글은 어떻게 보이나요?**
- **켜짐 상태**:
  - 청록-파랑 그라데이션 배경 (`#06b6d4 → #3b82f6 → #2563eb`)
  - 스위치가 오른쪽으로 이동 (22px)
  - 설정 메뉴에 "켜짐" 표시
- **꺼짐 상태**:
  - 회색 배경 (다크: `#4b5563`, 라이트: `#d1d5db`)
  - 스위치가 왼쪽에 위치
  - 설정 메뉴에 "꺼짐" 표시

**질문: 알림이 켜져 있으면 무엇을 받나요?**
- 챌린지 완료 알림
- 제로 챌린지 달성 알림
- 친구 요청 알림
- 환경 팁 알림
- 기타 중요한 앱 알림

## 5. 위치 설정

### 1. 위치 공유 동의
**질문: 위치 설정은 왜 필요한가요?**
- More 페이지의 제로웨이스트 맵 기능을 사용하기 위해 필요합니다
- 주변 제로웨이스트샵, 리필스테이션 등을 검색합니다

**질문: 위치 공유는 어떻게 설정하나요?**
1. 설정 메뉴에서 "위치" 항목을 클릭합니다
2. "위치 공유 동의" 토글 스위치를 클릭하여 켜거나 끕니다

**질문: 위치 토글은 어떻게 보이나요?**
- **켜짐 상태**:
  - 청록-파랑 그라데이션 배경
  - 스위치가 오른쪽으로 이동 (22px)
  - 설정 메뉴에 "켜짐" 표시
- **꺼짐 상태**:
  - 회색 배경
  - 스위치가 왼쪽에 위치
  - 설정 메뉴에 "꺼짐" 표시

**질문: 위치를 공유하지 않으면 어떻게 되나요?**
- 제로웨이스트 맵에서 "위치 설정을 켜면 주변 장소를 확인할 수 있습니다" 메시지가 표시됩니다
- 주변 장소 검색 기능을 사용할 수 없습니다

## 6. 화면 설정

### 1. 다크/라이트 모드
**질문: 화면 모드는 어떻게 변경하나요?**
1. 설정 메뉴에서 "화면" 항목을 클릭합니다
2. 원하는 모드를 선택합니다:
   - **라이트 모드**: 밝은 배경
   - **다크 모드**: 어두운 배경

**질문: 각 모드는 어떻게 표시되나요?**
- **라이트 모드**:
  - 태양 아이콘 (그라데이션)
  - 선택 시 청록색 테두리와 체크 아이콘
  - 흰색 배경, 검은색 텍스트

- **다크 모드**:
  - 달 아이콘 (그라데이션)
  - 선택 시 청록색 테두리와 체크 아이콘
  - 회색/검은색 배경, 흰색 텍스트

**질문: 다크 모드의 장점은?**
- 어두운 환경에서 눈의 피로를 줄입니다
- 배터리를 절약할 수 있습니다 (OLED 화면)

## 7. 색상 설정

### 1. 랭크 테마 선택
**질문: 색상 설정은 무엇인가요?**
- 앱의 강조 색상을 랭크별 테마로 변경할 수 있습니다
- 5가지 색상 테마가 있습니다

**질문: 어떤 색상 테마가 있나요?**
1. **기본 (다크/화이트)**:
   - 색상: `#e5e7eb` (라이트), `#374151` (다크)
   - 잠금 해제: 항상 사용 가능
   - 특징: 중립적인 회색 톤

2. **브론즈 (청록)**:
   - 색상: `#06b6d4` (청록색)
   - 잠금 해제: 기본 랭크에서 해제
   - 그라데이션: `#06b6d4 → #3b82f6 → #2563eb`

3. **실버 (민트)**:
   - 색상: `#14b8a6` (민트색)
   - 잠금 해제: 브론즈 랭크에서 해제
   - 그라데이션: `#14b8a6 → #10b981 → #059669`

4. **골드 (황금)**:
   - 색상: `#fcd34d` (황금색)
   - 잠금 해제: 실버 랭크에서 해제
   - 그라데이션: `#fde68a → #fcd34d → #fbbf24`

5. **플래티넘 (보라)**:
   - 색상: `#c084fc` (보라색)
   - 잠금 해제: 골드 랭크에서 해제
   - 그라데이션: `#c084fc → #a855f7 → #9333ea`

**질문: 색상 테마는 어떻게 변경하나요?**
1. 설정 메뉴에서 "색상" 항목을 클릭합니다
2. 원하는 색상 테마를 선택합니다
3. 잠금 해제된 테마만 선택할 수 있습니다

**질문: 잠긴 테마는 어떻게 표시되나요?**
- 블러 효과 (`backdrop-blur-sm`)와 반투명 검은 오버레이
- 자물쇠 아이콘 표시
- 투명도 50% (`opacity-50`)
- 클릭 시 "○○ 랭크에서 잠금해제" 경고 메시지

**질문: 선택된 테마는 어떻게 표시되나요?**
- 해당 랭크 색상의 테두리 (`borderColor: rank.color`)
- 해당 색상의 반투명 배경 (`${rank.color}15`)
- 그라데이션 체크 아이콘
- 랭크명과 설명이 해당 색상으로 표시

**질문: 색상 테마를 변경하면 어떻게 되나요?**
- 앱 전체의 강조 색상이 변경됩니다
- 버튼, 아이콘, 테두리 등의 색상이 적용됩니다
- 변경사항은 즉시 적용되고 자동으로 저장됩니다

## 8. 어항 설정

### 1. 어항 선택
**질문: 어항은 어떻게 변경하나요?**
1. Home 페이지 하단의 톱니바퀴(⚙️) 아이콘을 클릭합니다
2. "어항 설정" 화면이 열립니다
3. "어항 선택" 섹션에서 원하는 어항을 선택합니다

**질문: 어떤 어항을 선택할 수 있나요?**
- **기본 어항**: 항상 사용 가능
- **실버 어항**: 실버 랭크 보상을 수령해야 함
- **골드 어항**: 골드 랭크 보상을 수령해야 함
- **플래티넘 어항**: 플래티넘 랭크 보상을 수령해야 함

**질문: 잠긴 어항은 어떻게 표시되나요?**
- 회색 배경 (`bg-gray-100`)
- 투명도 50% (`opacity-50`)
- "랭킹 보상 필요" 빨간색 텍스트 (8px)
- 클릭 불가 (`cursor-not-allowed`)

**질문: 선택된 어항은 어떻게 표시되나요?**
- 청록색 테두리 (`border-cyan-500`)
- 밝은 청록색 배경 (`bg-cyan-50`)
- 어항 이름이 청록색으로 강조 (`text-cyan-600 font-medium`)

### 2. 물고기 설정
**질문: 물고기는 어떻게 설정하나요?**
1. 어항 설정의 "물고기" 섹션을 확인합니다
2. +/- 버튼으로 물고기 개수를 조절합니다
3. "랜덤 선택" 옵션을 켜면 자동으로 물고기가 배치됩니다
4. 랜덤 선택을 끄면 수동으로 물고기를 선택할 수 있습니다

**질문: 랜덤 선택이란?**
- **켜짐**:
  - 구매한 물고기 중 무작위로 선택된 물고기가 표시됩니다
  - +/- 버튼으로 개수만 조절합니다
  - 어떤 물고기가 표시될지는 랜덤입니다

- **꺼짐**:
  - 물고기를 수동으로 선택할 수 있습니다
  - 랭크별로 구매한 물고기가 표시됩니다
  - 원하는 물고기를 클릭하여 선택/해제합니다
  - 해제 시 자동으로 코리도라스가 선택됩니다 (없으면 첫 번째 물고기)

**질문: 물고기 개수 조절은 어떻게 되나요?**
- **랜덤 선택 ON**:
  - +/- 버튼으로 0 ~ 구매한 물고기 총 개수 범위로 조절
  - 예: 5마리 구매 시 0~5마리

- **랜덤 선택 OFF**:
  - 선택된 물고기 개수가 표시됩니다
  - + 버튼: 미선택 물고기 중 다음 물고기를 자동 추가
  - - 버튼: 마지막 선택 물고기를 제거
  - 최소 1마리는 선택되어야 합니다

**질문: 물고기 선택 화면은 어떻게 표시되나요?**
- 랭크별로 그룹화되어 표시됩니다 (브론즈, 실버, 골드, 플래티넘)
- 3열 그리드 레이아웃 (`grid-cols-3`)
- 각 물고기 카드:
  - 크기: 85px 높이
  - 물고기 아이콘: 32px
  - 물고기 이름: 10px
  - 선택 시: 청록색 테두리 + 배경, 이름이 청록색
  - 미선택 시: 기본 테두리, 회색 이름

**질문: 구매한 물고기가 없으면?**
- 보상 페이지에서 물고기를 먼저 구매해야 합니다
- 기본적으로 코리도라스 1마리가 제공됩니다

### 3. 장식품 설정
**질문: 장식품은 어떻게 설정하나요?**
1. 어항 설정의 "장식품" 섹션을 확인합니다
2. 구매한 장식품 목록이 표시됩니다
3. 원하는 장식품을 클릭하여 선택/해제합니다
4. 선택된 장식품만 어항에 표시됩니다

**질문: 장식품 개수는 제한이 있나요?**
- 구매한 장식품 개수만큼 모두 선택할 수 있습니다
- 최대 개수 제한은 없습니다
- "장식품 없애기" 버튼을 클릭하면 모든 장식품이 제거됩니다

**질문: 장식품 없애기는 어떻게 하나요?**
- "장식품 없애기" 체크박스를 클릭합니다
- 모든 선택된 장식품이 해제됩니다
- 어항이 깨끗해집니다

**질문: 장식품 선택 화면은 어떻게 표시되나요?**
- 랭크별로 그룹화되어 표시됩니다
- 3열 그리드 레이아웃 (`grid-cols-3`)
- 각 장식품 카드:
  - 크기: 85px 높이
  - 장식품 아이콘: 45px 영역
  - 장식품 이름: 10px
  - 선택 시: 청록색 테두리 + 배경, 이름이 청록색
  - 미선택 시: 기본 테두리, 회색 이름

**질문: 구매한 장식품이 없으면?**
- "보상 탭에서 장식품을 구매해주세요" 메시지가 표시됩니다
- 회색 배경의 안내 박스가 나타납니다

**질문: 장식품 위치는 어떻게 변경하나요?**
- 하단에 안내 메시지가 표시됩니다:
  > "어항 내에서 장식품을 두 번 터치하여 수정 모드로 진입 후 드래그하여 위치를 변경할 수 있습니다"
- Home 페이지에서 장식품을 더블클릭/더블탭합니다
- 설정 패널이 열리면 드래그하여 위치를 조정합니다
- 크기(50-150%)와 회전(0-360°)도 조절 가능합니다

### 4. 설정 적용
**질문: 설정을 저장하려면?**
- 하단의 "적용하기" 버튼을 클릭합니다

  - 하단에 고정 (`absolute bottom-0`)
- 클릭 시 설정이 저장되고 어항 설정이 닫힙니다

**질문: 적용하기를 누르지 않으면?**
- 변경사항이 저장되지 않을 수 있습니다
- 꼭 "적용하기" 버튼을 클릭하여 저장하세요

## 자주 묻는 질문 (FAQ)

**Q: 설정이 저장되지 않아요**
A: "적용하기" 버튼을 클릭했는지 확인하세요. 일부 설정은 자동 저장되지만, 어항 설정은 반드시 "적용하기"를 눌러야 합니다.

**Q: 색상 테마가 잠겨 있어요**
A: 해당 랭크에 도달하고 보상을 수령해야 잠금 해제됩니다. 챌린지를 완료하여 포인트를 획득하고 랭크를 올리세요.

**Q: 위치 공유를 켰는데도 지도가 안 나와요**
A: 브라우저 또는 앱의 위치 권한도 허용되어 있는지 확인하세요. 설정에서 위치 공유를 켜는 것 외에도 시스템 권한이 필요합니다.

**Q: 장식품이 너무 많아서 어항이 복잡해요**
A: 어항 설정에서 "장식품 없애기"를 클릭하여 모든 장식품을 제거하거나, 원하는 장식품만 선택하여 표시할 수 있습니다.

**Q: 영어 버전은 언제 나오나요?**
A: 현재 개발 중입니다. Coming Soon 상태이며, 출시 일정은 추후 공지될 예정입니다.

**Q: 다크 모드와 라이트 모드 중 어느 것을 사용해야 하나요?**
A: 개인 취향에 따라 선택하세요. 다크 모드는 어두운 환경에서 눈의 피로를 줄이고, 라이트 모드는 밝은 환경에서 가독성이 좋습니다.

**Q: 물고기가 한 마리만 표시돼요**
A: 어항 설정에서 물고기 개수를 조절하거나, 랜덤 선택을 켜서 더 많은 물고기를 표시할 수 있습니다. 물고기를 추가로 구매하면 더 다양하게 표시됩니다.

**Q: 로그아웃하면 데이터가 사라지나요?**
A: 저장된 데이터는 유지됩니다. 다시 로그인하면 모든 데이터를 확인할 수 있습니다.

**Q: 아이디를 변경했는데 중복 오류가 나요**
A: 다른 사용자가 이미 사용 중인 아이디입니다. 다른 아이디를 시도하거나 숫자나 언더스코어(_)를 추가하여 고유한 아이디를 만드세요.

**Q: 생년월일 입력 시 빨간색으로 표시돼요**
A: 올바른 날짜를 입력했는지 확인하세요. 월은 1~12, 일은 해당 월의 일수에 맞게 입력해야 하며, 미래 날짜는 입력할 수 없습니다. 2월은 28일 또는 29일까지만 입력 가능합니다.

**Q: 휴대폰 번호가 자동으로 형식화돼요**
A: 네, 정상입니다. 숫자만 입력하면 XXX-XXXX-XXXX 형식으로 자동 변환됩니다. 010-1234-5678 형태로 저장됩니다.

**Q: 프로필 사진이 업로드되지 않아요**
A: 이미지 파일 형식을 확인하세요 (JPG, PNG 권장). 파일 크기가 너무 크면 업로드가 실패할 수 있습니다. 네트워크 연결도 확인하세요.

**Q: 계정을 탈퇴하면 복구할 수 있나요?**
A: 아니요, 계정 탈퇴 시 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다. 신중하게 결정하세요.

**Q: 이메일 형식이 잘못되었다고 나와요**
A: 이메일은 example@domain.com 형식이어야 합니다. @와 도메인(.com, .kr 등)을 모두 포함해야 합니다.
