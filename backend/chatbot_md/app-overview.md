# Ecostep 앱 사용 가이드

## 앱 소개

**Ecostep**은 환경 보호 활동을 게임처럼 즐기면서 실천할 수 있는 친환경 습관 형성 앱입니다.

**핵심 컨셉:**
- 챌린지를 완료하여 포인트를 획득합니다
- 포인트로 물고기와 장식품을 구매합니다
- 나만의 어항을 꾸미면서 랭크를 올립니다
- 친구들과 랭킹을 비교하며 함께 환경을 보호합니다

---

## 핵심 개념

### 1. 포인트 시스템
**질문: 포인트는 무엇인가요?**
- 앱 내에서 사용하는 가상 화폐입니다
- 환경 보호 활동을 하면 포인트를 획득합니다
- 포인트로 물고기와 장식품을 구매할 수 있습니다

**질문: 포인트는 어떻게 확인하나요?**
- 보상 페이지 상단에 현재 보유 포인트가 표시됩니다
- 각 페이지에서도 확인할 수 있습니다

**질문: 포인트와 누적 포인트의 차이는?**
- **포인트**: 현재 사용 가능한 포인트 (아이템 구매 시 차감됨)
- **누적 포인트**: 지금까지 획득한 모든 포인트 (랭크 계산용, 소비해도 감소 안 함)
- 예: 10,000P를 획득하고 3,000P를 소비하면
  - 포인트: 7,000P (사용 가능)
  - 누적 포인트: 10,000P (랭크 계산)

### 2. 랭크 시스템
**질문: 랭크는 무엇인가요?**
- 누적 포인트에 따라 결정되는 등급입니다
- 랭크가 높을수록 더 많은 혜택을 받습니다

**질문: 랭크는 어떻게 구성되어 있나요?**
- **브론즈 (Bronze)**: 0~2,099 누적 포인트
- **실버 (Silver)**: 2,100~6,299 누적 포인트
- **골드 (Gold)**: 6,300~12,599 누적 포인트
- **플래티넘 (Platinum)**: 12,600 누적 포인트 이상

**질문: 랭크가 올라가면 무엇이 좋나요?**
- 고급 어항을 잠금 해제할 수 있습니다
- 더 특별한 물고기와 장식품을 구매할 수 있습니다
- 랭크별 색상 테마를 사용할 수 있습니다
- 커뮤니티에서 높은 순위를 차지합니다

**질문: 랭크가 내려갈 수 있나요?**
- 아니요, 누적 포인트는 감소하지 않으므로 랭크도 내려가지 않습니다
- 아이템을 구매해도 랭크에는 영향이 없습니다

### 3. 수질 시스템
**질문: 수질은 무엇인가요?**
- 어항의 깨끗한 정도를 나타내는 지표입니다
- 0%~100% 범위로 표시됩니다

**질문: 수질은 왜 중요한가요?**
- 수질이 낮으면 어항이 흐릿하게 보입니다
- 환경 보호 활동을 꾸준히 하고 있는지 보여줍니다
- 챌린지를 완료하면 수질이 100%로 회복됩니다

**질문: 수질은 어떻게 관리하나요?**
- 매일 챌린지를 완료하면 수질이 유지됩니다
- 챌린지를 하지 않으면 하루에 5%씩 감소합니다
- 수질 상태:
  - 80% 이상: 맑음 (파란색)
  - 50~79%: 보통 (노란색)
  - 50% 미만: 탁함 (빨간색)

## 앱 사용 흐름

Ecostep의 전체적인 사용 흐름은 다음과 같습니다:

```
1. 챌린지/환경 상식으로 포인트 획득
   ↓
2. 누적 포인트로 랭크 상승
   ↓
3. 보상 탭에서 물고기/장식품 구매
   ↓
4. 홈에서 어항 꾸미기
   ↓
5. 커뮤니티에서 친구와 랭킹 비교
   ↓
(반복)
```

### 단계 1: 포인트 획득하기

**질문: 포인트는 어떻게 얻나요?**

**방법 1: 챌린지 탭 - 주간 챌린지 완료**
- 챌린지 페이지로 이동합니다
- 월요일에 주간 챌린지를 선택합니다 (예: "텀블러 사용하기")
- 매일 "완료" 버튼을 클릭하여 챌린지를 완료합니다
- **보상**: 100P + 플라스틱 절약량 + 수질 100% 회복
- 자세한 내용은 [챌린지 가이드](challenge.md)를 참고하세요

**방법 2: 챌린지 탭 - 플라스틱 사용 추적**
- 챌린지 페이지의 "제로 챌린지" 탭으로 이동합니다
- 월요일에 주간 플라스틱 목표를 설정합니다 (예: 500g)
- 일주일 동안 사용한 플라스틱을 기록합니다
- 목표를 달성하면 월요일에 알림이 발송됩니다
- **보상**: 700P (달성률 1% 이상 시)
- 자세한 내용은 [챌린지 가이드 - 플라스틱 사용 추적](challenge.md#3-플라스틱-사용-추적-제로-챌린지)을 참고하세요

**방법 3: 기타 탭 - 오늘의 환경 상식**
- 기타(More) 페이지로 이동합니다
- "오늘의 환경 상식" 섹션을 찾습니다
- 팁을 펼치고 "확인(+100P)" 버튼을 클릭합니다
- **보상**: 100P (하루 1회)
- 자세한 내용은 [기타 가이드 - 오늘의 환경 상식](more.md#1-오늘의-환경-상식)을 참고하세요

**포인트 획득 요약:**
- 주간 챌린지 완료: 100P × 7일 = 700P/주
- 플라스틱 목표 달성: 700P/주
- 환경 상식 확인: 100P × 7일 = 700P/주
- **총 최대**: 2,100P/주

### 단계 2: 랭크 올리기

**질문: 랭크는 어떻게 올리나요?**

1. **포인트를 꾸준히 획득합니다**
   - 챌린지, 환경 상식으로 포인트를 모읍니다
   - 누적 포인트가 자동으로 증가합니다

2. **보상 페이지에서 진행 상황을 확인합니다**
   - 보상 탭으로 이동합니다
   - 상단에 현재 랭크와 진행도가 표시됩니다
   - 예: "실버까지 45% 남음"

3. **랭크 달성 시 보상을 수령합니다**
   - 실버, 골드, 플래티넘 랭크 달성 시 어항을 받을 수 있습니다
   - 어항 카드를 클릭하여 수령합니다
   - 수령한 어항은 자동으로 홈에 적용됩니다

**랭크별 소요 시간 (예상):**
- 브론즈 → 실버 (2,100P): 약 1주
- 실버 → 골드 (4,200P 추가): 약 2주
- 골드 → 플래티넘 (6,300P 추가): 약 3주

자세한 내용은 [보상 가이드 - 랭크 시스템](rewards.md#1-랭크-시스템)을 참고하세요

### 단계 3: 보상 탭에서 아이템 구매하기

**질문: 물고기와 장식품은 어떻게 구매하나요?**

1. **보상 페이지로 이동합니다**
   - 하단 내비게이션에서 "보상" 탭을 클릭합니다

2. **현재 포인트를 확인합니다**
   - 상단에 보유 포인트가 표시됩니다

3. **원하는 아이템을 선택합니다**
   - **물고기**: 어항에서 헤엄치는 물고기를 구매합니다
     - 브론즈: 200P~400P
     - 실버: 500P~700P
     - 골드: 1,000P~1,200P
     - 플래티넘: 1,500P~1,700P
   - **장식품**: 어항을 꾸미는 아이템을 구매합니다
     - 해초, 산호초, 바위 등 다양한 장식품

4. **아이템 카드를 클릭하여 구매합니다**
   - 포인트가 충분하면 즉시 구매됩니다
   - 포인트가 부족하면 "포인트 부족" 메시지가 표시됩니다

**주의사항:**
- 현재 랭크보다 높은 랭크의 아이템은 잠금 상태입니다
- 예: 브론즈 랭크에서는 실버, 골드, 플래티넘 아이템을 구매할 수 없습니다
- 구매한 아이템은 취소할 수 없으니 신중하게 선택하세요

자세한 내용은 [보상 가이드](rewards.md)를 참고하세요

### 단계 4: 홈에서 어항 꾸미기

**질문: 구매한 아이템은 어떻게 사용하나요?**

**물고기 설정:**
1. 홈 페이지의 어항 설정 버튼(톱니바퀴 아이콘)을 클릭합니다
2. "물고기 선택" 탭으로 이동합니다
3. 표시할 물고기를 선택합니다
   - 개별 선택: 원하는 물고기를 체크합니다
   - 랜덤 표시: 자동으로 물고기가 배치됩니다
4. 물고기 개수를 슬라이더로 조절합니다 (1~5마리)
5. "적용" 버튼을 클릭하여 저장합니다

**장식품 설정:**
1. 홈 페이지의 어항 설정에서 "장식품 선택" 탭으로 이동합니다
2. 표시할 장식품을 선택합니다
3. "적용" 버튼을 클릭합니다

**장식품 위치/크기/회전 조절:**
1. 어항에서 장식품을 **더블클릭** (PC) 또는 **더블탭** (모바일)합니다
2. 설정 패널이 하단에 나타납니다
3. 장식품을 드래그하여 위치를 변경합니다
4. 슬라이더를 움직여 크기(50%~150%)와 회전(0°~360°)을 조절합니다
5. "장식품 위치 저장" 버튼을 클릭하여 확정합니다

**어항 선택:**
- 랭크별 어항을 수령한 경우 어항 설정에서 변경할 수 있습니다
- 기본 어항, 실버 어항, 골드 어항, 플래티넘 어항

자세한 내용은 [홈 가이드](home.md)를 참고하세요

### 단계 5: 커뮤니티에서 친구와 랭킹 비교

**질문: 친구와 함께 사용하려면 어떻게 하나요?**

1. **커뮤니티 페이지로 이동합니다**
   - 하단 내비게이션에서 "커뮤니티" 탭을 클릭합니다

2. **친구를 추가합니다**
   - 초대 링크를 생성하여 친구에게 공유합니다
   - 친구가 링크로 가입하면 자동으로 친구 목록에 추가됩니다

3. **랭킹을 확인합니다**
   - 친구 랭킹: 친구들과의 순위를 비교합니다
   - 전체 랭킹: 모든 사용자와의 순위를 비교합니다
   - 순위는 누적 포인트로 결정됩니다

4. **친구의 어항을 구경합니다**
   - 친구 목록에서 친구를 클릭합니다
   - 친구의 어항과 통계를 확인할 수 있습니다

자세한 내용은 [커뮤니티 가이드](community.md)를 참고하세요

## 5개 탭 소개

Ecostep은 5개의 주요 탭으로 구성되어 있습니다:

### 1. 홈 (Home)
**주요 기능:**
- 나만의 어항 보기
- 물고기와 장식품 관리
- 수질 상태 확인
- 환경 통계 확인 (플라스틱 절약량, 나무 효과, 연속 달성)

**자세히 보기:** [홈 가이드](home.md)

### 2. 챌린지 (Challenge)
**주요 기능:**
- 주간 챌린지 선택 및 완료
- 플라스틱 사용량 추적 및 목표 설정
- 커스텀 챌린지 생성
- 챌린지 히스토리 확인

**포인트 획득:**
- 챌린지 완료: 100P/일
- 플라스틱 목표 달성: 700P/주

**자세히 보기:** [챌린지 가이드](challenge.md)

### 3. 보상 (Rewards)
**주요 기능:**
- 현재 랭크와 진행도 확인
- 물고기 구매 (200P~1,700P)
- 장식품 구매
- 랭크별 어항 보상 수령

**랭크 혜택:**
- 실버: 실버 어항, 실버 아이템 잠금 해제
- 골드: 골드 어항, 골드 아이템 잠금 해제
- 플래티넘: 플래티넘 어항, 플래티넘 아이템 잠금 해제

**자세히 보기:** [보상 가이드](rewards.md)

### 4. 커뮤니티 (Community)
**주요 기능:**
- 친구 초대 및 관리
- 친구 랭킹 확인
- 전체 랭킹 확인
- 친구 어항 구경하기

**순위 계산:**
- 누적 포인트로 순위가 결정됩니다
- 실시간으로 업데이트됩니다

**자세히 보기:** [커뮤니티 가이드](community.md)

### 5. 기타 (More)
**주요 기능:**
- 오늘의 환경 상식 (100P/일)
- 제로웨이스트 맵 (주변 친환경 장소 검색)
- 고객 센터 (AI 챗봇)
- 설정 (프로필, 언어, 알림, 위치, 화면, 색상)

**환경 상식 카테고리:**
- 랜덤, 재활용 팁, 생활 습관, 에너지 절약, 제로웨이스트

**자세히 보기:**
- [기타 가이드](more.md)
- [설정 가이드](settings.md)

## 게임 루프

Ecostep의 핵심 게임 루프는 다음과 같이 반복됩니다:

### 일일 루프 (매일)
```
1. 챌린지 완료 (+100P, 수질 회복)
2. 환경 상식 확인 (+100P)
3. 플라스틱 사용 기록
4. 어항 확인 및 꾸미기
   ↓
하루 최대 200P 획득
```

### 주간 루프 (월요일)
```
1. 새로운 주간 챌린지 선택
2. 플라스틱 목표 설정
3. 지난 주 플라스틱 목표 달성 시 보상 수령 (+700P)
   ↓
주간 목표 시작
```

### 랭크 상승 루프 (1~3주마다)
```
1. 누적 포인트로 랭크 상승
2. 새로운 어항 수령
3. 새로운 아이템 잠금 해제
4. 더 높은 랭크 목표 설정
   ↓
새로운 콘텐츠 즐기기
```

### 장기 목표
```
- 플래티넘 랭크 달성
- 모든 물고기와 장식품 수집
- 친구 랭킹 1위 달성
- 환경 보호 습관 형성
```

## 빠른 시작 가이드

### 첫날 해야 할 것
1. ✅ 챌린지 탭에서 주간 챌린지 선택 (월요일)
2. ✅ 챌린지 완료 (+100P)
3. ✅ 기타 탭에서 환경 상식 확인 (+100P)
4. ✅ 챌린지 탭에서 플라스틱 목표 설정 (월요일)
5. ✅ 보상 탭에서 첫 물고기 구매 (200P)
6. ✅ 홈에서 어항 확인

### 일주일 후
- 누적 포인트: 약 1,400P~2,100P
- 랭크: 브론즈 → 실버 근접
- 물고기 구매: 3~5마리
- 수질: 100% 유지 (매일 챌린지 완료 시)

### 한 달 후
- 누적 포인트: 약 6,000P~8,000P
- 랭크: 골드 달성
- 물고기 구매: 10마리 이상
- 장식품: 다양하게 배치
- 친구: 5명 이상 초대

## 자주 묻는 질문 (FAQ)

**Q: 하루에 몇 포인트를 얻을 수 있나요?**
A: 최대 200P입니다. 챌린지 완료 100P + 환경 상식 100P입니다. 추가로 주간 플라스틱 목표 달성 시 700P를 받을 수 있습니다.

**Q: 포인트를 사용하면 랭크가 내려가나요?**
A: 아니요. 랭크는 누적 포인트로 계산되며, 아이템을 구매해도 누적 포인트는 감소하지 않습니다.

**Q: 챌린지를 안 하면 어떻게 되나요?**
A: 수질이 하루에 5%씩 감소합니다. 수질이 낮으면 어항이 흐릿하게 보입니다. 챌린지를 완료하면 수질이 100%로 회복됩니다.

**Q: 플라스틱 목표를 달성하지 못하면 어떻게 되나요?**
A: 페널티는 없습니다. 달성률 1% 이상일 때만 700P 보상을 받습니다. 다음 주에 다시 도전하세요.

**Q: 친구를 초대하면 보상이 있나요?**
A: 현재는 친구 초대 보상이 없습니다. 하지만 친구와 함께 랭킹을 비교하며 더 재미있게 사용할 수 있습니다.

**Q: 구매한 아이템을 환불할 수 있나요?**
A: 아니요. 구매한 아이템은 환불할 수 없습니다. 신중하게 선택하세요.

**Q: 어항 보상을 받지 않으면 어떻게 되나요?**
A: 언제든지 받을 수 있습니다. 보상 페이지의 어항 카드를 클릭하여 수령하세요.

**Q: 환경 상식 카테고리를 변경하면 언제 적용되나요?**
A: 내일부터 적용됩니다. 환경 상식은 매일 자정에 생성되므로, 오늘은 기존 카테고리의 팁이 표시됩니다.

**Q: 물고기가 너무 많아서 어항이 복잡해요**
A: 어항 설정에서 물고기 개수를 줄이거나, 특정 물고기의 선택을 해제할 수 있습니다. 랜덤 표시를 사용하면 자동으로 적절한 수가 표시됩니다.

**Q: 장식품 위치를 저장하지 않고 닫으면 어떻게 되나요?**
A: 드래그로 이동한 위치는 자동으로 저장됩니다. 크기와 회전은 실시간으로 적용됩니다. "장식품 위치 저장" 버튼은 현재 설정을 확정할 때 사용합니다.

**Q: 플래티넘 랭크 이후에는 무엇을 하나요?**
A: 플래티넘은 최고 랭크입니다. 계속 포인트를 모아 더 많은 아이템을 구매하고, 친구와 랭킹을 경쟁하며, 어항을 꾸밀 수 있습니다. 환경 보호 습관을 유지하는 것이 가장 중요한 목표입니다!

## 팁과 전략

### 포인트 효율적으로 모으기
- 매일 챌린지와 환경 상식을 빠짐없이 완료하세요 (200P/일)
- 플라스틱 목표를 현실적으로 설정하여 매주 700P를 받으세요
- 수질을 100%로 유지하여 어항을 아름답게 관리하세요

### 랭크 빠르게 올리기
- 첫 주 목표: 실버 달성 (2,100P) → 매일 200P + 플라스틱 목표 700P = 2,100P
- 둘째 주 목표: 골드 근접 (4,200P 추가) → 2주간 총 4,200P
- 셋째 주 목표: 골드 달성 (6,300P)

### 아이템 구매 전략
- 초반에는 저렴한 브론즈 물고기 위주로 구매하세요 (200P~400P)
- 랭크가 올라가면 고급 물고기를 구매하세요
- 장식품은 포인트가 충분할 때 구매하세요
- 마음에 드는 아이템부터 구매하여 어항을 꾸미세요

### 어항 꾸미기 팁
- 장식품을 3개 이상 배치하여 다채롭게 꾸미세요
- 물고기 개수는 3~5마리가 적당합니다
- 장식품 크기와 회전을 조절하여 입체감을 만드세요
- 랭크별 어항을 수령하여 분위기를 바꿔보세요

### 커뮤니티 활용하기
- 친구를 초대하여 함께 환경 보호를 실천하세요
- 친구 랭킹을 확인하여 동기부여를 받으세요
- 친구의 어항을 구경하여 아이디어를 얻으세요
